Problem: Missing Positive Integer
Given an unsorted integer array A, find the smallest missing positive integer (≥ 1).

Examples:
A = [1, 2, 0] → Returns 3

A = [3, 4, -1, 1] → Returns 2

A = [7, 8, 9, 11, 12] → Returns 1

Constraints:
Time: O(N)

Space: O(1) (modify input array in-place if needed)
